Commands
All available chat & DM commands. Type them in the message box.
💬 Chat Commands
/helpShow all commands in chat
/me [action]Send an emote action
/shrugSend ¯\_(ツ)_/¯
/flipFlip a coin (heads/tails)
/tossSame as /flip
/roll [N]Roll a die 1–N (default 6)
/8ball [question]Ask the magic 8-ball
/countShow your message count
/clearClear your local chat view
👥 Social Commands
@[user] [message]Mention & highlight a user
/dm [user] [message]Send a quick DM from chat
/gift [user] [amount]Gift XP to another user
/trade [user]Open trade window with user
/ignore [user]Alias for /mute
/mute [user]Hide a user's messages (local)
/unmute [user]Stop hiding a user's messages
⚔ Clan Commands
/claninvite [user] [clan]Invite user to a clan by name or tag (owner/dev)
/invite [user] clanInvite user to your own clan (owner only)
/invite [user] [room]Invite user to a custom chat room
🏠 Room Commands
/pinPin your last message (room owner)
/unpinRemove pinned message (room owner)
/filter on|offToggle word censor (room owner)
/kick [user]Kick/ban user from this room (owner)
🛡 Dev / Mod Commands
/announce [text]System announcement (VIP/dev only)
/ban [user]Platform ban for 24h (dev only)
/ban [user] chatBan from current room (owner/dev)
/unban [user]Remove platform ban (dev only)
📩 DM Commands
/me [action]Emote in a DM
/shrugSend ¯\_(ツ)_/¯ in DM
/flipCoin flip in DM
/roll [N]Dice roll in DM
/8ball [question]8-ball in DM
/gift [user] [amount]Gift XP from a DM
/countYour message count in DM
